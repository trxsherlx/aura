# Aura's Privacy Policy

### By using Aura you are agreeing to be bound by these privacy policy.

1. What does it store?

- The Aura stores only the ID of the servers it is on along with the server language (EN-US or PT-BR).

2. Who can access the data?

- Aura developers only.

3. If you store data, how can users have that data removed?

- If you want the data to be removed or of any other reason, contact the Aura Developers. [Support Server](https://discord.gg/RFGhKkh9AD)

4. Why do you need the data?

- Without those most of the bot wouldn't be functional. The configuration system will not work because it is not customizable.

This policy is effective as of **November 8, 2020**.
